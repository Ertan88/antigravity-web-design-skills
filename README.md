# Antigravity Web Design Skills

This repository provides a fully automated, decision-tree-based skill framework
for Gemini Antigravity focused on professional web design workflows.

## Features
- AI Controller with decision tree logic
- Skill conflict resolution
- Quality scoring and telemetry
- One-click macOS installation

## Installation (macOS)
1. Download the repository as ZIP
2. Run install/install-mac.command
3. Restart Antigravity

## Usage
Just describe the website you want to build.
The agent will automatically select and orchestrate the correct skills.
