## Strategic Role
Guarantees inclusivity and legal compliance.

## Compliance Targets
- WCAG 2.1 AA minimum
- Keyboard-first navigation
- Screen reader compatibility

## Mandatory Checks

### 1. Visual Accessibility
- Minimum contrast ratio 4.5:1
- Never rely on color alone for meaning

### 2. Interaction Accessibility
- All actions must be keyboard accessible
- Focus states must be visible
- Logical tab order

### 3. Semantic Structure
- Use semantic HTML roles
- Avoid div-only layouts
- Label all form inputs

### 4. Cognitive Accessibility
- Avoid long unbroken text
- Clear error messages
- Predictable interactions

## Output Artifacts
- Accessibility compliance checklist
- Risk warnings for non-compliant elements

## Enforcement Rule
Accessibility decisions override visual aesthetics.
