## Strategic Role
Aligns content structure with search engine intent while preserving human readability.

## SEO Philosophy
SEO exists to serve users, not algorithms.
Never sacrifice clarity or accessibility for keyword density.

## Mandatory Workflow

### 1. Intent Mapping
For each page:
- Identify dominant intent:
  - Informational
  - Commercial
  - Transactional
- Reject mixed-intent pages

### 2. Keyword Architecture
- One primary keyword per page
- 3–5 semantic secondary terms
- Use entities, not repetitions

### 3. Structural SEO
- Exactly ONE H1 per page
- Logical heading nesting
- No skipped heading levels

### 4. Metadata Rules
- Title ≤ 60 chars, intent-driven
- Description ≤ 155 chars, CTA oriented
- No keyword stuffing

### 5. Internal Linking
- Use contextual anchors
- Avoid generic anchors ("click here")

## Output Artifacts
- SEO-optimized page outline
- Metadata recommendations
- Semantic keyword map

## Conflict Resolution
If SEO conflicts with UX or Accessibility:
→ SEO yields.
