## Strategic Role
Transforms strategy into intuitive user experiences.

## UX Principles

### 1. User Flow First
- Define entry → action → exit
- Remove unnecessary steps

### 2. Cognitive Load Reduction
- One primary action per screen
- Visual hierarchy guides attention

### 3. Consistency
- Reusable components
- Predictable behavior

## UI Rules
- Spacing > decoration
- Typography over graphics
- Clear affordances

## Output Artifacts
- UX flow map
- Component logic
- Interaction rules

## Enforcement
UX governs layout, visuals, and interactions.
